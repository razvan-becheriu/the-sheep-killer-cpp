
This is the C++ port of the original 'thesheepkiller' project developed by:

- Nicola {MSX} Lugato

- Kierren KAS Small

Website:

https://www.lugato.net/thesheepkiller/

The code has been generated using Google Gemini by translating the initial
sources from Pascal to C++.


to compile install dependencies:
 - SDL2_image
 - SDL2_mixer
 - GL
 - GLU
 - tinyxml2

and run:

make
